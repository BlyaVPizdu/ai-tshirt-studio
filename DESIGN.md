# AI T-Shirt Studio

## Goal

Production-ready SaaS for creating and ordering AI-generated T-shirt designs.

## Layout

- Header
- Left Prompt Panel (320px)
- Center Preview Panel (flex)
- Right Properties Panel (300px)
- Saved Designs section below

## Style

- Modern SaaS
- Similar to Canva, Figma, Linear
- White cards
- Light background
- Soft shadows
- Rounded corners
- Clear typography

## Primary Color

Indigo

## Components

### PromptPanel

- Prompt textarea
- Microphone
- Provider select
- Upload image
- Generate button

### PreviewPanel

- T-shirt preview
- Drag image
- Resize image
- Rotate image

### PropertiesPanel

- Size
- Rotation
- Position
- Save
- Export PNG

### SavedDesigns

- Preview
- Edit
- Delete
- Add to Cart

### Cart

- Quantity
- Remove
- Checkout
